# Pachax — Sistemas Digitales

Website oficial de Pachax. Software, automatización y marketing digital.

## Estructura del Proyecto

- `index.html`: Estructura y contenido SEO.
- `styles.css`: Diseño premium, animaciones y sistema de diseño.
- `script.js`: Interacciones, canvas de partículas y lógica de scroll.
- `assets/`: Imágenes y recursos gráficos.

## Despliegue

Este proyecto está preparado para ser desplegado en **Vercel** o **Netlify** conectando este repositorio de GitHub.
